📊 Loan Default Analysis Dashboard – Power BI Project
📌 Project Overview

  This project presents an interactive Loan Default Analysis Dashboard built using Power BI to analyze applicant demographics, financial behavior, loan             distribution, and default risk trends.
  
  The dashboard helps financial institutions and analysts identify:
  
  1.High-risk borrower segments.
  2.Loan distribution patterns.
  3.Employment and income-related risk factors
  4.Year-over-year loan and default changes
  5.Demographic influences on loan behavior
  
  The project combines data cleaning, transformation, DAX calculations, and visualization techniques to provide actionable business insights for loan risk            management.

🛠️ Tools & Technologies Used
  1.Power BI
  2.DAX (Data Analysis Expressions)
  3.Excel Dataset
  4.Data Modeling
  5.Data Visualization
